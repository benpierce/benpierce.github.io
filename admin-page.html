How to Lock the App Down by Jira Group

In the initial release, all users had access to use the User Activity Dashboard; however, the data returned by the App is scoped to only data that users have permission to see. That being said, we recognize that in certain cases, organizations may only want the User Activity Dashboard to be useable for Administrators or special groups (such as Engineering Managers).

Our latest addition allows Admins/App Admins to specify which groups should have access to run the User Activity Dashboard. The default is still to allow all users to use the App, but now that behavior can be overridden.

To lock User Activity Dashboard down to only specific groups, an Admin/App Admin can go into the "App" menu and click "Manage your apps".
